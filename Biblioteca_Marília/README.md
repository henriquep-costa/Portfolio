# 📚 Biblioteca Marília

Aplicação web simples feita com Flask para cadastro e listagem de livros.

## 🚀 Tecnologias

- Python
- Flask
- HTML

## ⚙️ Como executar

```bash
git clone https://github.com/seu-usuario/Portfolio.git
cd Portfolio/Biblioteca_Marília

python -m venv venv
venv\Scripts\activate  # Windows

pip install -r requirements.txt
python app.py

Acesse: http://127.0.0.1:5000
```

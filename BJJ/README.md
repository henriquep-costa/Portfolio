# BJJ Techniques API

Web application to manage Brazilian Jiu-Jitsu techniques using a REST API.

## Features

- List techniques
- Add new techniques
- Store data using SQLite
- Simple frontend consuming the API with JavaScript

## Technologies

- Python
- Flask
- SQLite
- HTML
- JavaScript

## Project structure

BJJ/
├── app.py
├── jiujitsu.db
├── templates/
│ └── index.html
└── README.md

## How to run locally

1. Install dependencies

```bash
pip install flask
Run the application

python app.py
Open in your browser

http://127.0.0.1:5000
Purpose
This project was created for learning backend development, REST APIs, and basic frontend integration using Python and Flask.
```

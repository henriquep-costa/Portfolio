# Cronograma de Estudos TJSP

Este projeto gera um **cronograma de estudos para concursos do TJSP** (Tribunal de Justiça de São Paulo).  
Ele organiza automaticamente as matérias em dias da semana, define o tempo de estudo por matéria e inclui a resolução de questões.  
O objetivo é fornecer uma forma prática e organizada de distribuir o estudo, ajudando a gerenciar o tempo e a disciplina durante a preparação para concursos.

---

## Objetivo do Projeto

- Criar um cronograma diário de estudo com **2 matérias por dia**.
- Distribuir **3 horas de estudo por dia**, dividindo igualmente entre as matérias.
- Definir um número fixo de questões para resolução por matéria (20 questões por padrão).
- Gerar um arquivo HTML (`cronograma.html`) para visualização fácil do cronograma.
- Tornar o cronograma **personalizável**, permitindo ajustes em matérias, número de horas ou quantidade de questões.

---

## Funcionalidades

- ✅ Divisão automática das matérias por dia da semana.
- ✅ Cálculo do tempo de estudo por matéria.
- ✅ Definição de número fixo de questões por matéria.
- ✅ Geração de arquivo HTML com todo o cronograma organizado.
- ✅ Personalização de:
  - Lista de matérias
  - Horas de estudo por dia
  - Número de matérias por dia
  - Número de questões por matéria

---

## Como usar

### 1. Preparação

1. Certifique-se de ter **Python 3** instalado.
2. Coloque o script `cronograma.py` na pasta do seu projeto.

### 2. Executar o script

No terminal, acesse a pasta do projeto e execute:

```bash
python cronograma.py
```

# Análise de Volume de Treinos

Projeto simples de análise de dados utilizando Python e Pandas para analisar o volume total de treinos ao longo do tempo.

## Objetivo

Analisar o volume total de treinos ao longo do tempo.

O volume foi calculado como:

carga × repetições × séries

## Tecnologias

- Python
- Pandas
- Jupyter Notebook

## Como executar

1. Clone o repositório

git clone https://github.com/seuusuario/analise-volume-treino

2. Instale as dependências

pip install pandas matplotlib

3. Abra o notebook

analise_treino.ipynb

## Análises realizadas

- Limpeza e tratamento de dados
- Cálculo do volume de treino
- Volume total por dia
- Comparação entre tipos de treino
